🔍 Regular Expressions (Regex)

Finding Patterns in Strings with JavaScript

Today's Goal

Regular Expressions (Regex) are a way to search for patterns inside text.

Instead of searching for one exact word, Regex lets us search for things like:

  • ✔ Email addresses
  • ✔ Phone numbers
  • ✔ ZIP Codes
  • ✔ Password requirements
  • ✔ Words containing certain letters
  • ✔ Numbers inside text

Regex Cheat Sheet

Basic Search

/cat/

Find the word "cat"

Ignore Capital Letters

/cat/i

Matches Cat, CAT, cat...

Find Every Match

/cat/g

Global Search

Character Ranges

[A-Z]

Capital letters

Numbers

[0-9]

Any digit

Word Characters

\w

Letters and numbers

Whitespace

\s

Spaces or tabs

Word Boundary

\b

Beginning or end of a word

JavaScript Methods

Method Purpose
.test() Returns true or false if a match exists.
.match() Returns an array of all matches.
.replace() Replaces matching text with something else.

Remember

😊 Nobody memorizes Regex.

Professional developers regularly use online tools to help write and test Regular Expressions.

The important part is understanding that Regex searches for patterns, not just exact words.
